# Агент-ориентированная модель формирования инфляционных ожиданий домохозяйств

Проект реализует агент-ориентированную модель формирования инфляционных ожиданий домохозяйств в заданной инфляционной среде. Инфляция по укрупненным товарным группам задается экзогенно, а эндогенными результатами являются ожидаемая инфляция, воспринимаемая инфляция и гетерогенность ожиданий между домохозяйствами.

## Состав проекта

`settings.py` — параметры модели, proxy-группы домохозяйств и пути к данным;
`agents.py` — классы центрального банка и домохозяйств;
`runner.py` — запуск baseline, сценарных экспериментов и расчет метрик;
`plots2.py` — построение всех графиков.

## Данные

Модель ищет обработанные данные в папке `data/processed/`:

```text
data/processed/v1_inflation_paths.json
data/processed/v1_monthly_dataset.csv
```

`v1_inflation_paths.json` содержит ряды инфляции по трем группам:

```json
{
  "food": [0.05, 0.06],
  "nonfood": [0.04, 0.05],
  "services": [0.06, 0.07]
}
```

`v1_monthly_dataset.csv` используется для сопоставления модели с опросными данными:

```text
date,real_expected_inflation,real_perceived_inflation,aggregate_inflation,key_rate,inflation_target
```

## Агенты

### Центральный банк

Центральный банк хранит инфляционный таргет, веса товарных групп и формирует официальный сигнал:

```python
signal = signal_alpha * target + (1 - signal_alpha) * aggregate_inflation
```

Чем выше `signal_alpha`, тем ближе сигнал к таргету. Чем ниже `signal_alpha`, тем сильнее сигнал следует за текущей инфляцией.

### Домохозяйства

В модели есть три proxy социально-доходные группы:

| Тип | Интерпретация |
|---|---|
| `low_income` | низкодоходные / ценочувствительные |
| `middle_income` | среднедоходные / сбалансированные |
| `high_income` | высокодоходные / финансово включенные |

Доход не моделируется как денежный поток. Группы различаются косвенно: структурой корзины, заметностью товарных групп, частотой обновления ожиданий, весом официального сигнала, чувствительностью и памятью воспринимаемой инфляции.

Текущие параметры групп:

| Параметр | `low_income` | `middle_income` | `high_income` |
|---|---:|---:|---:|
| Доля группы | 0.30 | 0.50 | 0.20 |
| Корзина: food | 0.47 | 0.39 | 0.24 |
| Корзина: nonfood | 0.25 | 0.35 | 0.33 |
| Корзина: services | 0.28 | 0.26 | 0.43 |
| Заметность: food | 0.68 | 0.55 | 0.42 |
| Заметность: nonfood | 0.18 | 0.25 | 0.25 |
| Заметность: services | 0.14 | 0.20 | 0.33 |
| Частота обновления | 0.30 | 0.40 | 0.52 |
| Вес сигнала ЦБ | 0.12 | 0.25 | 0.38 |
| Чувствительность восприятия | 1.45 | 1.30 | 1.18 |
| Bias восприятия | 0.040 | 0.035 | 0.030 |
| Память восприятия | 0.58 | 0.50 | 0.42 |

## Шаг модели

На каждом периоде:

1. Модель получает инфляцию по группам `food`, `nonfood`, `services`.
2. Центральный банк рассчитывает агрегированную инфляцию и сигнал.
3. Домохозяйство рассчитывает инфляцию своей корзины.
4. Если включена заметность цен, используется `salience_weights`; иначе — `basket_weights`.
5. Формируется текущая воспринимаемая инфляция.
6. Воспринимаемая инфляция сглаживается через память.
7. Формируется информационный якорь ожиданий:

```python
anchor = personal_weight * perceived + cb_weight * cb_signal + social_weight * neighbor_expectation
```

8. Домохозяйство обновляет ожидание с вероятностью `update_frequency`.
9. Модель сохраняет агрегированные и групповые показатели.

## Baseline

Baseline — это полная спецификация с тремя proxy-группами, личным ценовым опытом, заметностью цен, памятью восприятия и сигналом ЦБ. В baseline отключены социальная сеть и обратная связь ожиданий в инфляцию:

```python
use_social_network = False
expectations_to_inflation_strength = 0.0
```

Эти каналы анализируются отдельно в сценарных экспериментах.

## Сценарии

### H1. Информационные ограничения

Проверяется роль частоты обновления ожиданий:

- `no_update`;
- `slow_update`;
- `baseline_update`;
- `fast_update`.

### H2. Личный ценовой опыт

Проверка устроена как декомпозиция:

- `homogeneous_agents` — одинаковые агенты;
- `basket_only` — различаются только корзины;
- `basket_salience` — различаются корзины и заметность цен;
- `behavior_only` — корзины одинаковые, но различаются поведенческие параметры;
- `full_income_proxy_profile` — полный proxy-профиль.

Во всех H2-сценариях отключены социальная сеть и обратная связь ожиданий с инфляцией, а начальные ожидания фиксируются. Это изолирует вклад личного ценового опыта.

### H3. Коммуникация ЦБ

Проверяется роль веса официального сигнала и формы сигнала:

- `no_trust`;
- `low_trust`;
- `baseline_trust`;
- `high_trust`;
- `target_signal`;
- `inflation_signal`.

### H4. Социальная сеть

Проверяется влияние структуры социальной сети:

- `no_network`;
- `random_network`;
- `homophilic_network`;
- `echo_chamber_network`;
- `mixed_high_influence_network`.

## Основные показатели

В истории и метриках сохраняются:

- `mean_expectation` — средние инфляционные ожидания;
- `mean_perceived_inflation` — воспринимаемая инфляция;
- `mean_price_experience_inflation` — личный ценовой опыт;
- `std_expectation` — разброс ожиданий внутри населения;
- `between_type_disagreement` — разрыв ожиданий между группами;
- `between_type_perceived_gap` — разрыв воспринимаемой инфляции;
- `between_type_price_experience_gap` — разрыв личного ценового опыта;
- `update_share` — доля обновивших ожидания;
- `mean_personal_weight`, `mean_cb_weight`, `mean_social_weight` — веса информационных каналов.
